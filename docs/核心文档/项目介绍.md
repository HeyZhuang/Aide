# 🎨 Jaaz AI 设计助手 - 项目介绍

## 📖 项目概述

**Jaaz** 是一款基于 AI 的本地桌面设计助手应用，集成了多种 AI 图像生成和编辑功能，为设计师提供智能化的设计工作流。

### 核心特性

- 🤖 **AI 图像生成** - 支持多种 AI 模型（Gemini, OpenAI, Midjourney 等）
- 🎨 **PSD 文件处理** - 导入、编辑、导出 Photoshop 文件
- 🖼️ **画布编辑** - 直观的图层管理和元素编辑
- 💬 **智能对话** - 基于 LangGraph 的多智能体对话系统
- 📦 **模板管理** - 内置设计模板库
- 🔐 **用户系统** - 支持本地和 Google OAuth 登录

## 🚀 核心功能模块

### 1. 图像生成模块

- **Text-to-Image** - 文本描述生成图片
- **Image-to-Image** - 图片风格转换
- **多模型支持** - 灵活切换不同 AI 模型
- **批量生成** - 支持批量任务处理

### 2. 画布编辑模块

- **图层管理** - 完整的图层增删改查
- **元素操作** - 移动、缩放、旋转、裁剪
- **PSD 支持** - 导入导出 Photoshop 文件
- **自动布局** - 智能元素定位

### 3. 对话系统模块

- **多智能体** - Planner + Image Creator 协作
- **工具调用** - 自动选择合适的 AI 工具
- **上下文管理** - 完整的对话历史记录
- **流式响应** - 实时反馈生成进度

### 4. 用户系统模块

- **本地认证** - SQLite 用户数据库
- **Google OAuth** - 第三方登录
- **权限管理** - 游客、用户、管理员角色
- **会话管理** - JWT Token 认证
